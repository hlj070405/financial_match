# Vue 3 前端启动指南

## 快速启动（3 步）

### 1. 安装依赖

```bash
cd frontend-vue
npm install
```

或使用 pnpm（更快）：
```bash
pnpm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问：http://localhost:3000

## 项目特性

✅ **Vue 3 Composition API** - 现代化的 Vue 开发方式
✅ **Vite 超快构建** - 秒级热更新
✅ **TailwindCSS** - 保持原有美观样式
✅ **组件化架构** - 易于维护和扩展
✅ **TypeScript 就绪** - 可随时升级到 TS

## 核心组件

- `UploadArea.vue` - 文件上传（拖拽、进度条）
- `ResultDisplay.vue` - 结果展示（加载状态、分析结果）
- `ChartDisplay.vue` - ECharts 图表
- `App.vue` - 主应用组件

## API 代理

开发环境下，`/api` 请求会自动代理到 `http://localhost:8000`

## 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录，可直接部署到 Nginx 等服务器。

## 常见问题

**Q: 安装依赖失败？**
```bash
# 清除缓存重试
npm cache clean --force
npm install
```

**Q: 端口被占用？**
修改 `vite.config.js` 中的 `server.port`

**Q: 样式不生效？**
确保 TailwindCSS 已正确安装并配置
