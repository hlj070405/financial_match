## ⚠️ 常见问题

**Q: 上传失败？**
- 确保 Dify 服务正在运行（http://localhost）
- 检查 PDF 文件大小不超过 15MB

**Q: 分析失败？**
- 检查后端服务是否正常运行
- 查看后端控制台的错误信息
- 确认 API Key 是否正确

**Q: 前端无法连接后端？**
- 确保后端运行在 http://localhost:8000
- 检查浏览器控制台的错误信息（F12）

## 📊 API 接口说明

### 上传 PDF
```
POST http://localhost:8000/api/upload-pdf
Content-Type: multipart/form-data
Body: file=<PDF文件>
```

### 分析文档
```
POST http://localhost:8000/api/analyze
Content-Type: application/json
Body: {
  "file_id": "文件ID",
  "question": "分析问题",
  "style": "专业分析"
}
```
