# 财报抓取微服务使用指南

## 📋 功能概述

这是一个智能财报PDF抓取微服务，具有以下特点：

### ✨ 核心功能

1. **智能企业识别**
   - ✅ 支持企业全称：比亚迪股份有限公司
   - ✅ 支持企业简称：比亚迪
   - ✅ 支持拼音：biyadi、byd
   - ✅ 支持股票代码：002594
   - ✅ 模糊匹配，容错率高

2. **行业批量抓取**
   - ✅ 识别行业关键词：新能源汽车、半导体、医药等
   - ✅ 自动获取该行业所有上市公司
   - ✅ 批量下载财报（可限制数量）

3. **多源数据抓取**
   - ✅ 优先从巨潮资讯网（官方披露平台）
   - ✅ 备用东方财富网
   - ✅ 自动重试和容错

4. **高鲁棒性**
   - ✅ 多重匹配策略
   - ✅ 相似度排序
   - ✅ 详细错误信息

## 📡 API使用示例

### API 1: 抓取财报

**端点**: `POST /api/financial-reports/fetch`

**请求示例**：

```json
{
  "query": "比亚迪",
  "year": 2023
}
```

**响应示例**：

```json
{
  "type": "single",
  "companies": [
    {
      "code": "002594",
      "name": "比亚迪",
      "similarity": 0.9
    }
  ],
  "pdf_files": [
    "./financial_reports/002594_2023_annual_report.pdf"
  ],
  "errors": []
}
```

### API 2: 测试企业匹配

**端点**: `GET /api/financial-reports/test-match/{query}`

**示例**: `GET /api/financial-reports/test-match/byd`

**响应**：

```json
{
  "query": "byd",
  "matches": [
    {
      "code": "002594",
      "name": "比亚迪",
      "pinyin": "biyadi",
      "short_pinyin": "byd",
      "similarity": 0.9
    }
  ],
  "count": 1
}
```

### API 3: 测试行业识别

**端点**: `GET /api/financial-reports/test-industry/{query}`

**示例**: `GET /api/financial-reports/test-industry/新能源汽车`

**响应**：

```json
{
  "query": "新能源汽车",
  "industry": "新能源汽车",
  "companies": [
    {"code": "002594", "name": "比亚迪"},
    {"code": "300750", "name": "宁德时代"},
    ...
  ],
  "total_count": 50
}
```

fetchReport('比亚迪')

## ⚠️ 注意事项

### 1. 巨潮资讯网API

巨潮资讯网的API可能会变化，如果下载失败，需要：
1. 访问 http://www.cninfo.com.cn/
2. 打开浏览器开发者工具
3. 查看网络请求，更新API端点

### 2. 反爬虫策略

如果遇到反爬虫限制：
- 添加请求头伪装
- 增加请求间隔
- 使用代理IP

```python
# 添加请求头
headers = {
    'User-Agent': 'Mozilla/5.0 ...',
    'Referer': 'http://www.cninfo.com.cn/'
}
```

### 3. 并发控制

批量下载时建议控制并发数：

```python
# 使用信号量限制并发
semaphore = asyncio.Semaphore(3)  # 最多3个并发

async def download_with_limit(stock_code):
    async with semaphore:
        return await crawler.download_report(stock_code)
```

## 📞 集成到现有系统

### 在ChatModule中使用

```vue
<script setup>
const fetchAndAnalyze = async (companyName) => {
  // 1. 抓取财报
  const reportResult = await axios.post('/api/financial-reports/fetch', {
    query: companyName,
    year: 2023
  })
  
  if (reportResult.data.pdf_files.length > 0) {
    // 2. 上传PDF到Dify
    const pdfPath = reportResult.data.pdf_files[0]
    
    // 3. 调用分析接口
    const analysisResult = await axios.post('/api/chat', {
      message: `分析这份财报`,
      files: [{ path: pdfPath, type: 'document' }]
    })
    
    return analysisResult.data
  }
}
</script>
```
