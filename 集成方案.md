# TrendRadar + 财报分析系统 集成方案

## 📋 项目概览

你现在有两个系统：

1. **PDF RAG 财报分析系统** (`pdf-rag-system/`)
   - 基于 Dify 的 RAG 系统
   - 上传 PDF 财报，AI 智能分析
   - 前端：Vue.js + TailwindCSS
   - 后端：FastAPI + Dify API

2. **TrendRadar 舆情监控系统** (`hotspot/`)
   - 多平台热点聚合（微博、知乎、抖音等）
   - 关键词精准筛选
   - 本地 SQLite 数据库存储
   - HTML 报告生成

## 🎯 集成目标

**为财报分析系统增加实时舆情监控能力**，形成"财报数据 + 市场舆情"的双重分析维度。

### 方案二：数据联动 - 智能舆情关联 ⭐⭐ 推荐

**难度**: ⭐⭐⭐☆☆  
**开发时间**: 1-2 天  
**适用场景**: 自动化关联，提升用户体验

#### 实现步骤

1. **后端添加舆情查询接口**
   ```python
   # pdf-rag-system/backend/main.py
   
   import sqlite3
   from datetime import datetime
   
   @app.get("/api/sentiment/{company_name}")
   async def get_company_sentiment(company_name: str):
       """查询公司相关舆情"""
       today = datetime.now().strftime("%Y-%m-%d")
       db_path = f"../hotspot/output/news/{today}.db"
       
       conn = sqlite3.connect(db_path)
       cursor = conn.cursor()
       
       # 查询包含公司名的新闻
       cursor.execute("""
           SELECT title, platform, rank, hot, url, timestamp
           FROM news 
           WHERE title LIKE ?
           ORDER BY timestamp DESC
           LIMIT 20
       """, (f'%{company_name}%',))
       
       results = cursor.fetchall()
       conn.close()
       
       return {
           "company": company_name,
           "news_count": len(results),
           "news": [
               {
                   "title": row[0],
                   "platform": row[1],
                   "rank": row[2],
                   "hot": row[3],
                   "url": row[4],
                   "time": row[5]
               }
               for row in results
           ]
       }
   ```

2. **前端调用接口**
   ```javascript
   // frontend-vue/src/api/sentiment.js
   
   export function getCompanySentiment(companyName) {
     return request({
       url: `/api/sentiment/${companyName}`,
       method: 'get'
     })
   }
   ```

3. **在分析结果页面显示舆情**
   ```vue

           <a :href="news.url" target="_blank">
             {{ news.title }}
           </a>
           <span class="platform">{{ news.platform }}</span>
           <span class="rank">排名: {{ news.rank }}</span>
         </div>
       </div>
     </div>
   </template>
   
   <script>
   import { getCompanySentiment } from '@/api/sentiment'
   
   export default {
     data() {
       return {
         sentimentData: null
       }
     },
     methods: {
       async loadSentiment(companyName) {
         this.sentimentData = await getCompanySentiment(companyName)
       }
     }
   }
   </script>
   ```

4. **自动提取公司名**

## 💡 关键技术点

### 1. 公司名提取
```python
import re

def extract_company_name(text):
    """从财报中提取公司名"""
    # 方法1：正则匹配
    pattern = r'([\u4e00-\u9fa5]+(?:公司|集团|股份))'
    match = re.search(pattern, text)
    if match:
        return match.group(1)
    
    # 方法2：使用 NER（更准确）
    # 可以用 spaCy 或其他 NER 工具
    return None
```

### 2. SQLite 数据查询
```python
def query_news_by_keyword(keyword, days=7):
    """查询最近 N 天包含关键词的新闻"""
    from datetime import datetime, timedelta
    
    results = []
    for i in range(days):
        date = (datetime.now() - timedelta(days=i)).strftime("%Y-%m-%d")
        db_path = f"output/news/{date}.db"
        
        if os.path.exists(db_path):
            conn = sqlite3.connect(db_path)
            cursor = conn.cursor()
            cursor.execute(
                "SELECT * FROM news WHERE title LIKE ?",
                (f'%{keyword}%',)
            )
            results.extend(cursor.fetchall())
            conn.close()
    
    return results
```
